# Contributing

Bedankt voor je interesse in **Awesome Schuldhulp NL**!

## Wat we wel/niet accepteren

**Wel:**
- Officiële instanties, brancheverenigingen, helplines (NL).
- Onderzoek, datasets, en wetenschappelijke publicaties.
- Open-source software relevant voor schuldhulp/budget/financiën.
- SaaS-platformen met duidelijke functionele meerwaarde voor inwoners,
  professionals of gemeenten.

**Niet:**
- Affiliate-links of "lead-magnet" gated content zonder publieke waarde.
- Snelle-lening-producten of incassobureaus zonder NVVK/WCK-borging.
- Persoonlijke blogs zonder substantiële, terugkerende waarde.

## Format

```
- [Naam](https://url) — Korte beschrijving (één zin).
```

- Alfabetisch binnen sectie.
- Eén link per regel.
- Beschrijving in dezelfde taal als de sectie-stijl.

## PR-checklist

- [ ] Link werkt en is HTTPS.
- [ ] In de juiste sectie geplaatst.
- [ ] Alfabetische volgorde behouden.
- [ ] Lint-check: `awesome-lint README.md` (optioneel).

## Code of Conduct

We volgen de [Contributor Covenant](https://www.contributor-covenant.org/).
Wees vriendelijk; we bouwen iets voor mensen in geldzorgen — empathie is de
basislaag.
